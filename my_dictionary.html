Vocabulary plus

Vocabulary plus is a mobile app for keeping track of new words that you come across.


Features


  • Manage words for multiple languages
  • Import words from local memory/sd card
  • Import words from server
  • Supports JSON and text file format

Why I built this app?


To increase my vocabulary!! I like learning new languages. I know Konkani, English, Hindi, Marathi, a bit of Kannada and a bit of Telgu as well.

I wanted to increase my English vocabulary and learn some new words in Telgu. I'm not good at remembering words so I decided to create an app which will help me in keeping track of new words that I have learnt.

I'm doing quite good in learning Telgu and my next target is Tamil and Spanish!!
